安装 Hexo

2016-04-01 #hexo #tool

  1. 安装 Node.js
  2. 安装 Hexo
1$ sudo npm install -g hexo-cli
  1. 初始化 Hexo
1$ hexo init <folder>
2$ cd <folder>
3$ npm install
  1. 生成静态页面
1$ hexo generate
  1. 在 Github 新建项目
  2. 填写网站及Git配置信息 _config.yml
1deploy:
2    type: git
3    repo: https://github.com/koulerz/kouler.git
  1. 安装 hexo-deployer-git
1$ npm install hexo-deployer-git --save
  1. 部署项目到 Github
1$ hexo deploy
