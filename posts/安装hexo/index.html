安装 Hexo

Apr 1, 2016 #hexo #tool

  1. 安装 Node.js

  2. 安装 Hexo

sudo npm install -g hexo-cli
  1. 初始化 Hexo
hexo init <folder>
cd <folder>
npm install
  1. 生成静态页面
hexo generate

  1. 在 Github 新建项目

  2. 填写网站及Git配置信息 _config.yml

deploy:
    type: git
    repo: https://github.com/koulerz/kouler.git
  1. 安装 hexo-deployer-git
$ npm install hexo-deployer-git --save
  1. 部署项目到 Github
hexo deploy
