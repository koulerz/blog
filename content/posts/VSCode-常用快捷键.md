## macOS

### Editor

```
Command + N             新建 Tab
Command + Shift + N     新建窗口
Ctrl + `                显示/隐藏控制台
Command + B             显示/隐藏侧边栏
Command + \             分割编辑器组
Command + 1/2/3         定位编辑器组
Command + P             查找文件/安装插件
Option + Shift + F      格式化文档
Ctrl + R                显示最近打开过的文件
Option + Z              切换自动换行
```

### File

```
Command + K R           在 Finder 中打开当前文件
Command + K P           复制当前文件路径
Command + K M           更改文件语言

Command + Option + ↑    向上插入光标
Command + Option + ↓    向下插入光标
Option + Shift + I      在选中行的末尾插入光标

Command + Shift + K     删除当前行
Option + ↑              向上移动当前行
Option + ↓              向下移动当前行
Option + Shift + ↑      向上复制当前行
Option + Shift + ↓      向下复制当前行
Command + Enter         向下插入行
Command + Shift + Enter 向上插入行

Option + Shift + →      按块选中
Option + Shift + ←      按块撤销选中
Command + D             选中当前选中内容的下一个
Command + L             选中当前行
Command + F2            选中所有当前单词
Command + Shift + L     选中所有当前内容

Ctrl + G                定位到指定行
Command + P + :         定位到指定行
Command + P + @         定位到对应 Symbols
Command + Shift + \     定位到代码块前后位置
Command + U             定位上一次光标操作
Command + K Command + D 定位到下一个选中内容的位置
```

## Windows

### Editor

```
Ctrl + N                新建 Tab
Ctrl + Shift + N        新建窗口
Ctrl + `                显示/隐藏控制台
Ctrl + B                显示/隐藏侧边栏
Ctrl + \                分割编辑器组
Ctrl + 1/2/3            定位编辑器组
Ctrl + P                查找文件/安装插件
Ctrl + Shift + F        格式化文档
Ctrl + C                显示最近打开过的文件
Alt + Z                 切换自动换行
```

### File

```
Ctrl + ↑                向上滚动一行
Ctrl + ↓                向下滚动一行

Ctrl + K P              复制当前文件路径
Ctrl + K M              更改文件语言

Ctrl + Alt + ↑          向上插入光标
Ctrl + Alt + ↓          向下插入光标
Alt + Shift + I         在选中行的末尾插入光标

Ctrl + Shift + K        删除当前行
Alt + ↑                 向上移动当前行
Alt + ↓                 向下移动当前行
Alt + Shift + ↑         向上复制当前行
Alt + Shift + ↓         向下复制当前行
Ctrl + Enter            向下插入行
Ctrl + Shift + Enter    向上插入行

Alt + Shift + →         按块选中
Alt + Shift + ←         按块撤销选中
Ctrl + F2               选中所有当前单词
Ctrl + D                选中当前选中内容的下一个
Ctrl + L                选中当前行
Ctrl + Shift + L        选中所有选中的内容

Ctrl + G                定位到指定行
Ctrl + P + :            定位到指定行
Ctrl + P + @            定位到对应 Symbols
Alt + →                 定位上一次光标位置
Alt + ←                 定位下一次光标位置
Ctrl + Shift + \        定位到代码块前后位置
F3                      定位到下一个选中内容的位置
Shift + F3              定位到上一个选中内容的位置
Ctrl + U                定位上一次光标操作
Ctrl + K Ctrl + D       定位到下一个选中内容的位置
```
